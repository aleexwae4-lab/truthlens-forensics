# TRUTHLENS FORENSICS PRO 🛡️

**Autonomous Forensic Intelligence Core v4.2**

TruthLens Forensics Pro is a high-performance, autonomous forensic analysis engine designed to decode deception in text, audio, and visual evidence. Built for peritial-grade investigations, it leverages advanced neural engines to identify manipulation, gaslighting, and psychological pressure with surgical precision.

## 🚀 God Mode Features

- **Multimodal Analysis:** Process raw text, PDF documents, high-resolution captures, and audio files.
- **30-Point Forensic Matrix:** Rigorous application of 30 specific indicators of deception.
- **Lie Triangle Analysis:** Granular scoring across Fact (Information), Time (Consistency), and Emotion (Congruence).
- **Bento-Grid Dashboard:** High-impact visual representation of credibility scores, risk levels, and heatmaps.
- **Peritial Reports:** One-click export to professional PDF forensic reports.
- **Neural Engine:** Powered by Gemini 3.1 Pro for state-of-the-art linguistic and cognitive analysis.

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router) / React 19
- **Styling:** Tailwind CSS 4.0 (Hardware/Editorial Aesthetic)
- **Animations:** Motion (Framer Motion)
- **Data Viz:** Recharts (Radar & Heatmaps)
- **Intelligence:** Google Gemini 3.1 Pro API
- **Reporting:** jsPDF & AutoTable

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/truthlens-forensics.git
   cd truthlens-forensics
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
